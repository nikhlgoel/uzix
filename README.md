# Uzix

Open-source multilingual prompt injection detector for AI systems (English, Hindi, Hinglish).

## What is this?
A dev-focused tool to spot prompt injection attacks in AI apps, with a special focus on Indian languages. Not fancy, just works.

- Rule-based and ML-based detection
- Dataset included (English, Hindi, Hinglish)
- API and browser extension planned

See `docs/` for tech/design notes, PRD, and more.
